Functions

Call any function from the extensive standard library from your Markdown code.

.somefunction {argument1} {argument2}
    Body argument
Or define your own:
```markdown
.function {greet}
    to from:
    **Hello, .to** from .from!

.greet {world} from:{Giorgio}
```


.text {**Hello, world** from Giorgio!} size:{small}

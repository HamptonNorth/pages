# Using images markdown

The syntax is:
```markdown
![Alt text](/media/start/redmug_logo_313x316.png)
```
This will be rendered as:
![Alt text](/media/start/redmug_logo_316x316.png)

Here is another example. Oronsay is a small headland, east and slightly north of Tobermory. A great anchorage. 
```markdown
![Sunset at Oronsay near Tobermory](/media/start/sunset_near_oronsay.png)
```
This will be rendered as:
![Sunset at Oronsay near Tobermory](/media/start/sunset_near_oronsay.png)

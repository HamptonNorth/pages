A home page for testing the bun-starter code
The stack to build bunstarter is:
  • Bun
  • Lit (reduce boilerplate for components)
  • SQLite for persistent storage
  • TailwindCSS
The Bun app is built as a combination static file server and API server using bunserve(). The TailwindCSS colours used have are coded as a custom theme with a primary and secondary pallette along with custom error colours and highlights.
Releases tagged 0.1.4 and above in gitHub, add authentication using the better-auth library.
The readme.md file contains install and set up notes. It will open in a new tab.
